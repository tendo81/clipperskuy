TS Telebot Apprem adalah bot Telegram otomatis untuk penjualan aplikasi premium, license key, dan produk non-fisik lainnya. Dilengkapi dengan pembayaran QRIS realtime, pengelolaan produk lengkap, sistem tiket CS, fitur affiliate, dan panel admin yang mudah digunakan. Dibangun dengan Node.js + Telegraf + MongoDB.
📦 Sistem Produk ▼
  • Tambah / edit / hapus kategori
  • Tambah / edit / hapus varian produk
  • Sorting kategori A–Z otomatis
  • Stok otomatis (auto berkurang setelah beli)
  • SNK (Syarat & Ketentuan) multiline
💳 Pembayaran QRIS Realtime ▼
  • Generate QRIS otomatis via Tokopay
  • Auto polling status pembayaran
  • Auto cancel setelah 5 menit timeout
  • Update saldo otomatis setelah bayar
  • Log transaksi ke channel Telegram
🛒 Sistem Order ▼
  • Cek stok otomatis sebelum order
  • Auto-delivery akun / license key
  • Simpan semua transaksi ke database
  • Riwayat transaksi lengkap per user
🎫 Tiket Customer Service ▼
  • User buat tiket CS multi-line
  • Admin balas tiket langsung dari panel
  • Admin tutup tiket via tombol
  • Filter tiket: Open / Pending / Closed
  • Delete tiket satuan atau semua
📊 Polling System ▼
  • Broadcast polling ke semua user
  • Hasil polling digabung realtime
  • Laporan polling ringkas otomatis
  • Hapus polling satuan atau semua
🤝 Affiliate System ▼
  • Komisi global (berlaku semua produk)
  • Komisi per produk (custom setiap item)
  • Komisi per user (custom setiap affiliate)
  • Auto-hitung komisi per transaksi
👥 User & Admin Panel ▼
  • Auto-register user saat pertama /start
  • Panel admin lengkap dengan inline keyboard
  • Saldo & deposit management
  • Notifikasi log ke channel
⚙️ Requirements ▼
🟢 Node.js 16+
📦 NPM / Yarn
🍃 MongoDB
🤖 Bot Token
💳 Tokopay API
📢 Channel Log
