📖 Panduan ClipperSkuy

Panduan lengkap dari instalasi sampai menghasilkan clip viral. Estimasi waktu baca: 10 menit.

📦 Instalasi

ClipperSkuy adalah aplikasi desktop Windows yang berjalan 100% lokal di PC kamu. Tidak perlu upload video ke cloud.

System Requirements

Komponen Minimum Recommended
OS Windows 10 (64-bit) Windows 11
RAM 8 GB 16 GB
Storage 2 GB free 10 GB+ free
GPU Tidak wajib NVIDIA/AMD/Intel (untuk rendering cepat)
Internet Untuk download YouTube & API —

Langkah Instalasi

  1. Download file installer dari halaman Download
  2. Extract file ZIP ke folder yang diinginkan
  3. Jalankan ClipperSkuy.exe
  4. Aplikasi akan otomatis membuat folder data di %APPDATA%\clipperskuy\data
💡
Trial Otomatis: Saat pertama kali membuka aplikasi, kamu otomatis mendapat 7 hari trial dengan semua fitur premium terbuka. Tidak perlu registrasi!

🔑 Setup API Key

ClipperSkuy menggunakan AI untuk menganalisa video dan menemukan momen viral. Kamu perlu API key gratis dari salah satu provider berikut:

Groq API Key (Recommended)

  1. Buka console.groq.com
  2. Buat akun gratis (login dengan Google)
  3. Klik "API Keys" → "Create API Key"
  4. Copy key yang diawali gsk_
  5. Paste di Settings → AI Configuration → Groq API Keys
🔄
Multi-Key Rotation: Tambahkan beberapa API key untuk auto-rotation. Saat satu key kena rate limit, key berikutnya otomatis digunakan.

Gemini API Key (Alternatif)

  1. Buka aistudio.google.com/apikey
  2. Login dengan akun Google
  3. Klik "Create API Key"
  4. Copy key yang diawali AIzaSy
  5. Paste di Settings → AI Configuration → Gemini API Keys
⚠️
Penting: Minimal satu API key (Groq atau Gemini) harus diisi agar AI clip detection berfungsi. Tanpa API key, kamu tetap bisa memotong video secara manual.

🎬 Membuat Project Pertama

Ada dua cara untuk membuat project baru — download dari YouTube atau upload video langsung dari PC.

📥 Download dari YouTube

  1. Buka halaman New Project (tombol + di sidebar)
  2. Paste URL YouTube di field yang tersedia
  3. Atur settings: Platform target, Reframing mode, Durasi clip
  4. Klik "Upload & Generate Clips"
  5. Tunggu proses download + analisis AI (progress ditampilkan realtime)
Contoh URL https://www.youtube.com/watch?v=dQw4w9WgXcQ
https://youtu.be/dQw4w9WgXcQ

📤 Upload Video dari PC

  1. Buka halaman New Project
  2. Drag & drop file video ke area upload, atau klik untuk browse
  3. Format yang didukung: MP4, MOV, AVI, MKV, WebM
  4. Ukuran maksimal: 5 GB
  5. Atur settings dan klik "Upload & Generate Clips"

✂️ Review & Edit Clips

Setelah AI selesai menganalisa, kamu akan melihat daftar clips yang ditemukan. Setiap clip memiliki:

  • Viral Score — Skor 0-100 yang menunjukkan potensi viral
  • Title — Judul yang di-generate AI berdasarkan konten
  • Duration — Panjang clip (bisa diedit)
  • Transcript — Teks transkripsi dari audio
  • Hook Text — Kalimat pembuka yang menarik perhatian

Klik pada clip untuk membuka Clip Editor — di sini kamu bisa:

  • Preview clip dengan subtitle
  • Edit start/end time
  • Ubah subtitle text dan styling
  • Pilih reframing mode
  • Generate hook text alternatif
  • Lihat rekomendasi platform terbaik

🎯 Reframing Mode

ClipperSkuy menawarkan 6 mode reframing untuk mengubah video landscape (16:9) ke portrait (9:16):

Mode Deskripsi Tier
Center Crop Crop tengah sederhana. Cepat dan reliable. Free
Fit (Blur) Full video + blur background di atas/bawah. Tidak ada bagian yang terpotong. Free
Face Track AI mengikuti wajah speaker secara dinamis. Smooth tracking sepanjang clip. Trial
Face + Blur Face tracking + blur background. Look paling profesional. Pro
Split Screen Speaker di atas, konten/slide di bawah. Cocok untuk tutorial. Free
Podcast Auto-detect 1 atau 2 speaker. Dynamic split screen untuk interview/podcast. Free

💬 Subtitle & Captions

Subtitle dihasilkan menggunakan Whisper AI dan di-burn langsung ke video saat render.

Fitur Subtitle

  • Word-by-word animation — Highlight kata saat diucapkan
  • Multi-bahasa — ID, EN, AR, ZH, JA, KO, dan lainnya
  • Custom styling — Font, warna, ukuran, posisi, outline
  • Hook text overlay — Teks menarik di 3 detik pertama
💡
Tips: Gunakan YouTube caption sebagai sumber subtitle jika tersedia — lebih cepat dan hemat API quota dibanding re-transcribe.

🎞️ Render & Export

Setelah puas dengan editing, klik tombol Render untuk menghasilkan file video final.

Proses Render

  1. ClipperSkuy akan menerapkan semua efek: reframing, subtitle, hook text, progress bar
  2. Progress ditampilkan secara realtime via WebSocket
  3. File output disimpan di folder project
  4. Kamu bisa render satu clip atau batch render semua clips sekaligus

Quality Preset

⚡
GPU Acceleration: Jika kamu punya GPU NVIDIA, AMD, atau Intel, aktifkan di Settings untuk rendering hingga 5x lebih cepat. ClipperSkuy otomatis mendeteksi hardware terbaik.

⚙️ Settings

Halaman Settings memungkinkan kamu mengkonfigurasi:

  • AI Configuration — API keys untuk Groq, Gemini, dan Pexels (B-Roll)
  • Video Processing — GPU acceleration, encoder, quality preset, resolusi output
  • Audio Enhancement — Noise reduction, voice clarity (Pro)
  • Retention Progress Bar — Warna, posisi, tinggi, opacity
  • Watermark & Branding — Logo, teks, animasi watermark
  • Output Folder — Lokasi penyimpanan default

🔧 Troubleshooting

Aplikasi tidak bisa dibuka

  • Pastikan Windows 10/11 64-bit
  • Jalankan sebagai Administrator
  • Periksa apakah antivirus memblokir aplikasi

AI clip detection gagal

  • Pastikan API key sudah diisi di Settings
  • Cek validitas API key (klik tombol refresh ↻)
  • Pastikan koneksi internet aktif
  • Coba ganti provider (Groq ↔ Gemini)

Render gagal atau video corrupt

  • Pastikan FFmpeg terinstall (biasanya bundled otomatis)
  • Coba ubah encoder ke CPU - libx264 (Software) di Settings
  • Kurangi resolusi dari 1080p ke 720p
  • Pastikan ada cukup ruang disk

YouTube download gagal

  • Pastikan yt-dlp terinstall
  • Update yt-dlp: pip install -U yt-dlp
  • Periksa apakah URL valid
  • Beberapa video mungkin diblokir oleh region/copyright

❓ FAQ Teknis

Di mana data disimpan?

Semua data tersimpan di %APPDATA%\clipperskuy\data\. Ini termasuk database, video source, dan clip output.

Bisa pakai API key gratis?

Ya! Groq dan Gemini menawarkan tier gratis yang cukup untuk penggunaan normal. Groq memberikan ~14.400 request/hari dan Gemini memberikan 15 request/menit.

Apakah aman untuk video pribadi?

100% aman. Video tidak pernah di-upload ke server manapun. Semua proses AI (kecuali analisis teks oleh Groq/Gemini) berjalan lokal. Yang dikirim ke API hanya teks transkripsi, bukan video.

Bisa digunakan untuk video selain YouTube?

Tentu! Kamu bisa upload video apapun dari PC (MP4, MOV, AVI, MKV, WebM) — tidak terbatas pada YouTube.

Bagaimana cara update?

Download versi terbaru dari website dan extract ke folder yang sama. Database dan settings akan tetap tersimpan karena disimpan di folder %APPDATA% yang terpisah.
