📖 ClipperSkuy Guide

Panduan lengkap dari instalasi sampai menghasilkan clip viral. Estimasi waktu baca: 10 menit.

📦 Installation

ClipperSkuy adalah aplikasi desktop Windows yang berjalan 100% lokal di PC kamu. Tidak perlu upload video ke cloud.

System Requirements

Component Minimum Recommended
OS Windows 10 (64-bit) Windows 11
RAM 8 GB 16 GB
Storage 2 GB free 10 GB+ free
GPU Not required NVIDIA/AMD/Intel (untuk rendering cepat)
Internet For YouTube downloads & API —

Installation Steps

  1. Download the installer from the Download
  2. Extract the ZIP file to your desired folder
  3. Run ClipperSkuy.exe
  4. The app will automatically create a data folder at %APPDATA%\clipperskuy\data
💡
Trial Otomatis: Saat pertama kali membuka aplikasi, kamu otomatis mendapat 7 hari trial dengan semua fitur premium terbuka. Tidak perlu registrasi!

🔑 API Key Setup

ClipperSkuy menggunakan AI untuk menganalisa video dan menemukan momen viral. Kamu perlu API key gratis dari salah satu provider berikut:

Groq API Key (Recommended)

  1. Go to console.groq.com
  2. Create a free account (login with Google)
  3. Click "API Keys" → "Create API Key"
  4. Copy the key starting with gsk_
  5. Paste in Settings → AI Configuration → Groq API Keys
🔄
Multi-Key Rotation: Tambahkan beberapa API key untuk auto-rotation. Saat satu key kena rate limit, key berikutnya otomatis digunakan.

Gemini API Key (Alternative)

  1. Go to aistudio.google.com/apikey
  2. Login with your Google account
  3. Click "Create API Key"
  4. Copy the key starting with AIzaSy
  5. Paste in Settings → AI Configuration → Gemini API Keys
⚠️
Penting: Minimal satu API key (Groq atau Gemini) harus diisi agar AI clip detection berfungsi. Tanpa API key, kamu tetap bisa memotong video secara manual.

🎬 Creating Your First Project

There are two ways to create a new project — download from YouTube or upload a video directly from your PC.

📥 Download from YouTube

  1. Open the New Project page (+ button in sidebar)
  2. Paste the YouTube URL in the provided field
  3. Configure settings: Target platform, Reframing mode, Clip duration
  4. Click "Upload & Generate Clips"
  5. Wait for download + AI analysis (progress shown in realtime)
Example URLs https://www.youtube.com/watch?v=dQw4w9WgXcQ
https://youtu.be/dQw4w9WgXcQ

📤 Upload Video from PC

  1. Open the New Project page
  2. Drag & drop video files to the upload area, or click to browse
  3. Supported formats: MP4, MOV, AVI, MKV, WebM
  4. Maximum size: 5 GB
  5. Configure settings and click "Upload & Generate Clips"

✂️ Review & Edit Clips

After AI finishes analyzing, you'll see a list of detected clips. Each clip has:

  • Viral Score — 0-100 score indicating viral potential
  • Title — AI-generated title based on content
  • Duration — Clip length (editable)
  • Transcript — Transcribed text from audio
  • Hook Text — Attention-grabbing opening text

Click on a clip to open the Clip Editor — here you can:

  • Preview clips with subtitles
  • Edit start/end time
  • Change subtitle text and styling
  • Choose reframing mode
  • Generate alternative hook text
  • View best platform recommendations

🎯 Reframing Mode

ClipperSkuy offers 6 reframing modes to convert landscape (16:9) video to portrait (9:16):

Mode Description Tier
Center Crop Simple center crop. Fast and reliable. Free
Fit (Blur) Full video + blur background on top/bottom. Nothing gets cropped. Free
Face Track AI dynamically follows the speaker's face. Smooth tracking throughout the clip. Trial
Face + Blur Face tracking + blur background. The most professional look. Pro
Split Screen Speaker on top, content/slides on bottom. Great for tutorials. Free
Podcast Auto-detect 1 or 2 speakers. Dynamic split screen for interviews/podcasts. Free

💬 Subtitle & Captions

Subtitles are generated using Whisper AI and burned directly into the video during rendering.

Subtitle Features

  • Word-by-word animation — Highlight words as they're spoken
  • Multi-language — ID, EN, AR, ZH, JA, KO, and more
  • Custom styling — Font, color, size, position, outline
  • Hook text overlay — Engaging text in the first 3 seconds
💡
Tips: Gunakan YouTube caption sebagai sumber subtitle jika tersedia — lebih cepat dan hemat API quota dibanding re-transcribe.

🎞️ Render & Export

When you're happy with your edits, click the Render button to generate the final video file.

Render Process

  1. ClipperSkuy will apply all effects: reframing, subtitles, hook text, progress bar
  2. Progress is shown in realtime via WebSocket
  3. Output files are saved in the project folder
  4. You can render a single clip or batch render all clips at once

⚙️ Settings

The Settings page allows you to configure:

  • AI Configuration — API keys for Groq, Gemini, and Pexels (B-Roll)
  • Video Processing — GPU acceleration, encoder, quality preset, output resolution
  • Audio Enhancement — Noise reduction, voice clarity (Pro)
  • Retention Progress Bar — Color, position, height, opacity
  • Watermark & Branding — Logo, text, watermark animation
  • Output Folder — Default save location

🔧 Troubleshooting

App won't open

  • Make sure you have Windows 10/11 64-bit
  • Run as Administrator
  • Check if antivirus is blocking the app

AI clip detection fails

  • Make sure API key is set in Settings
  • Check API key validity (click refresh ↻ button)
  • Make sure internet connection is active
  • Try switching providers (Groq ↔ Gemini)

Render fails or video is corrupt

  • Make sure FFmpeg is installed (usually bundled automatically)
  • Try changing encoder to CPU - libx264 (Software) in Settings
  • Reduce resolution from 1080p to 720p
  • Make sure there's enough disk space

YouTube download fails

  • Make sure yt-dlp is installed
  • Update yt-dlp: pip install -U yt-dlp
  • Check if the URL is valid
  • Some videos may be blocked by region/copyright

❓ Technical FAQ

Where is data stored?

All data is stored in %APPDATA%\clipperskuy\data\. Ini termasuk database, video source, dan clip output.

Can I use a free API key?

Ya! Groq dan Gemini menawarkan tier gratis yang cukup untuk penggunaan normal. Groq memberikan ~14.400 request/hari dan Gemini memberikan 15 request/menit.

Is it safe for private videos?

100% aman. Video tidak pernah di-upload ke server manapun. Semua proses AI (kecuali analisis teks oleh Groq/Gemini) berjalan lokal. Yang dikirim ke API hanya teks transkripsi, bukan video.

Can I use it with non-YouTube videos?

Of course! You can upload any video from your PC (MP4, MOV, AVI, MKV, WebM) — not limited to YouTube.

How do I update?

Download the latest version from the website and extract to the same folder. Database and settings will be preserved as they are stored in a separate %APPDATA% folder.
